A search engine hit calculator with a .NET backend and an Angular frontend

## Prerequisites

- .NET SDK
- Node.js and npm

## Instructions

In order to build and run the project, execute these from root

```bash
git clone https://github.com/maxdeekay/searchengine_techcheck.git
cd searchengine_techcheck\voyadosearchengine.client
npm install
cd searchengine_techcheck\VoyadoSearchEngine.Server
dotnet restore
dotnet build
dotnet run
```
